# Matreshka
Тест для обучения

Мишка
Уронили мишку на пол,
Оторвали мишке лапу.
Все равно его не брошу –
Потому что он хороший

Кораблик
Матросская шапка,
Веревка в руке,
Тяну я кораблик
По быстрой реке,
И скачут лягушки
За мной по пятам
И просят меня:
– Прокати, капитан!

Лошадка
Я люблю свою лошадку,
Причешу ей шёрстку гладко,
Гребешком приглажу хвостик
И верхом поеду в гости
